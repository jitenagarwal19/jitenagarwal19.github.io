React Native App To Manage Expense

May 30, 2016    react-native mobile-application hobby

Framework React-Native

I created this app when I started learning Javascript and explored different frameworks in Javascript, then I bumped into react-native and was really amazed by facebook’s approach of creating HYBRID Mobile App. React-Native is build around the philosophy that learn once and write everywhere over write once and run everywhere(philosophy of phonegap).

You might have to write more code uncommon code for Android and iOS in React-Native but you will be writing in same language(i.e. JS) and everything will be rendered natively when you will the app. I was at that time working on an Android Application and thought of writing a simple app on this platform.

App Money Management

Github Link
So the reasons why I built this app are -

  1. I wanted to learn react-native
  2. I wanted to build a simple app to track my expenses.

I felt the need to create my own expense management app because all the expense management apps out there are complex and do many things, which i didnt want. So I figured out building a very simple app that will do the job. So my basic requirements were very simple -

  1. Should have some fields to enter my expenses - (Name of expense, expense, and some description)
  2. Should have categories to store my expenses(Food, Travel, Shopping, Home and Other)
  3. Way to view the expenses on the basis of categories and time(monthly or yearly)
  4. Store the expenses locally.

Further Improvements that I will implement in the future -

  1. Adding OCR to scan bill images directly in the app.
  2. Storing data in the cloud (like firebase)
  3. Implementing User login and rollout public version of the app.
