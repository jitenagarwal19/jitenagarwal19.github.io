Million Words Social Network built using Ionic Framework

August 3, 2016    ionic-framework angularjs javascript mobile-application hobby

I was looking forward to learn more javascript frameworks after getting a glimpse of React-Native and basic understanding of javascript this time my focus was to learn a framwork that will help me in learning web-development too, So I came across AngularJs. After learning and understanding basics of AngularJs I was thinking of a cool project that I can do to make my understanding more robust. I thought of creating a simple Social Network type application which I could implement in AngularJs. I decided to go mobile again and was looking for a framework which was using AngularJs at its core to build a Hybrid Mobile Appilication. After searching a bit I came across Ionic Framework.

Ionic Framework

Ionic is the open source mobile sdk that will allow you to build Progressive Web and Native Apps easily. Features Of Ionic Framework -

  1. Integeration with cordova.
  2. Uses AngularJs
  3. Provides extensive set of components like list, buttons, label, header, footer, tab-navigation etc.
  4. Good Command Line Interface for debugging and building application(I used iOS and android)
  5. Supports SaSS and Less for themeing

The App - Million Words.

I like the concept of Sex of Ideas here’s the ted talk where in you take mate two ideas and derive a new idea out of it. So here the Million Words app was the child of Quora(where you ask questions) and Instagram(where you post pictures with some description). Yes, you guessed it right in Million Words you can ask questions that can be only answered using pictures and short description. And the name million words because A picture is worth million words Following were the features I implemented in the first version of Million words.

  1. Multi User
    • login
    • signup
    • User can add profile pictures and update their info(name, email, short description)
    • Other users can view profiles(Info, question, answers) of the users.
  2. Ask Question
    • A user can ask question.
    • Upvote and downvote a question.
    • Separate Screen for unanswered questions.
  3. Answer to a Question
    • User can answer the question by uploading a picture(from gallery or camera) along with short description.
    • User can comment on any answer
    • User can like the answer.
    • User can view other user’s answer

Libraries and Technologies Used -

  1. Ionic Framework v1 - To build the app.
  2. Cordova Library - Bridge between the native and Web comes with basic native functionality like logging, toast etc. I added Dialog and Camera plugin too.
  3. Grunt - To build and run tasks
  4. NPM - To add and update ionic and cordova plugins.
  5. Bower - To add libraries for the app like moments, firebase etc.
  6. Firebase - Google’s Awesome Realtime database which was backend of my application. I used Authentication and Database of the Firebase more about firebase here – –
  7. Git - I learned git a lot in this project becuase I made myself use branching for a feature. So it was fun.
  8. Geny Motion - Best Android Emulator I came across. I am mentioning this because it was a big relief for me when I came across Geny Motion, it’s fast and simple to use.

There are many other things I need to implement in this app, I will make it more polished in the future if anyone likes the idea and is willing to contribute please let me know. I haven’t made it open source yet.
